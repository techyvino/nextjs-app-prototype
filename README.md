## Getting Started
This is the Next js project builded in top of Next UI based tailwind css, code formatting with prettier and eslint for better and clean errorless code.

React: v18 
Next Js: v14
Tailwind CSS : v3.3.5



Requirements:
node: 8.18.0 or later,
pnpm: 8.10.0 or later,

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Second build the Production build

```bash
npm run build
# or
yarn run build
# or
pnpm run build
# or
bun run build
```

Third preview the Production build

```bash
npm start
# or
yarn start
# or
pnpm start
# or
bun start
```
